First Project Breakdown

To keep myself focused and in sync with progress I break the project into this fractions, and use the oportunity to make a blog post:

  1. Scheming the layout
    1. Draw a sample how the page would look like on smaller device
    2. Draw a sample how the page would look like on medium device
    3. Draw a sample how the page would look like on bigger device
  2. Making HTML code
    1. Make HTML file
    2. Make CSS file and create a helper code(boxes for header, section, aside and footer)
  3. Create grid layout
    1. Create grid for what is inside the container
    2. Change the grid for medium width
    3. Change the grid for big width
    4. Make corrections if needed
  4. Making flex layout
    1. Make a helper CSS for children of header items
    2. Set flex inside CSS file for all widths for header items
    3. Make a helper CSS for children of section items
    4. Set flex inside CSS file for all widths for section items
    5. Make a grid for section items
    6. Change layout for other widths
    7. Make a helper CSS for children of aside items
    8. Set flex inside CSS file for all widths for aside items
    9. Make correction if needed
  5. Styling the page
    1. Make another css file
    2. Style backgrounds, fonts, colors, buttons...
  6. Make media querries
    1. Make a file for responsive design
    2. Fine style for small devices
    3. Fine style for small-medium devices
    4. Fine style for medium devices
    5. Fine style for medium-big devices
    6. Fine style for big devices
    7. Fine style for even bigger devices
  7. Finish the project
    1. Insert pictures in HTML
    2. Rewrite sample text
    3. Check for bugs and finalize
